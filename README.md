## ✨ Features

- **Premium UI/UX:** A stunning, Apple-inspired glassmorphism design with soft ambient shadows and smooth hover interactions.
- **Dynamic Animations:** Integrated `AOS` (Animate On Scroll) and custom CSS keyframes for a lively, engaging user experience.
- **Responsive Design:** Flawless rendering across all devices, from mobile phones to ultra-wide desktop monitors.
- **Modern Product Cards:** Inline cart buttons, beautiful thumbnails, and dynamic stock indicators wrapped in frosted glass.
- **Robust Error Handling:** Safely handles incomplete backend data with graceful fallbacks.

## 🛠️ Technology Stack

### Frontend
- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Typography:** Plus Jakarta Sans (Google Fonts)

### Backend API
- **Platform:** Vercel / Node.js
- **Endpoints:** RESTful API for Products, Orders, Categories, and Reviews
